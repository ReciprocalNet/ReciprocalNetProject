1st Workshop
Reciprocal Net

Bloomington, Indiana, USA
March 5 - 6, 2005
Agenda

We will review the current state of hard and software and discuss future development needs.


Topics
  • Presentations on database structure, web interfaces and applications, associated software (e.g., Shortep, Cifimporter etc.)
  • Hands-on workshop utilizing web applications and associated software
  • Release schedule
  • Discussion on future governance of the project
